# ⌨️ Master Typing

A fast and interactive typing practice app designed to help you improve your typing speed and accuracy. Whether you're a beginner or looking to sharpen your skills, Master Typing gives you a clean and focused environment to practice every day.

## ✨ Features

- 🚀 **Real-time WPM tracking** — See your words per minute update as you type
- ✅ **Accuracy measurement** — Track how precise your typing is after each test
- ⏱️ **Timed typing tests** — Practice under time pressure to build speed
- 🔁 **Instant restart** — Reset and try again with a single click
- 📊 **Results summary** — Get your WPM and accuracy at the end of each session
- 🎯 **Random text prompts** — Fresh passages every round to keep practice varied
- 💻 **Clean minimal UI** — Distraction-free interface so you can focus on typing
- 📱 **Responsive design** — Works on desktop and mobile browsers
