# ⌨️ Typing Speed Test Master

A professional, feature-rich typing speed test application with 7 beautifully designed screens and smooth animations. Test and improve your typing skills with real-time feedback, detailed analytics, and competitive leaderboards.

## ✨ Features

- **7 Interactive Screens**: Splash/Intro, Dashboard, Test Settings, Gameplay, Pause Menu, Results Analytics, Leaderboard
- **Real-time Typing Feedback**: Character-by-character highlighting (green for correct, red wavy for errors)
- **Live Statistics**: WPM, Accuracy, Error count, Keystrokes, Consistency score
- **Customizable Tests**: Choose time duration (15s-120s), text type, difficulty level
- **Detailed Analytics**: Performance graphs, error heatmaps, progress tracking
- **Leaderboard System**: Global rankings and personal history with localStorage
- **Smooth Animations**: Screen transitions, button effects, counting numbers, confetti on high scores
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop
- **Dark Theme**: Beautiful deep navy color scheme with glassmorphism effects

## 🎯 Perfect For

- Typing practice and skill improvement
- Coding bootcamp projects
- JavaScript animation demonstrations
- Portfolio showcase

## 🛠️ Built With

- HTML5
- CSS3 (Animations, Flexbox, Grid, Glassmorphism)
- Vanilla JavaScript (No frameworks)
- LocalStorage API

## 📱 Screens Included

1. **Splash Screen** - 3D rotating logo, loading animation
2. **Dashboard** - User stats, progress chart, menu options
3. **Test Settings** - Customizable duration, text type, difficulty
4. **Gameplay** - Real-time typing with character highlighting
5. **Pause Menu** - Blur effect, resume with countdown
6. **Results** - Detailed analytics, confetti celebration
7. **Leaderboard** - Global rankings, personal history
